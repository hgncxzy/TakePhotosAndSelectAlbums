这个例子实现了以下小功能模块：

1. 底部弹窗实现选择 拍照/选择相册 的 ActionSheet UI. 

2. 使用 EasyPermission 框架处理权限请求的必要逻辑，非常好用，特别是针对拒绝权限的情况。

3. 通过原生相机拍照并显示，文件存储和相机拍照适配了 Android10 和 Android7。

4. 通过系统相册选择招聘并显示。
